# Iris dataset (demo)

This is a condensed 30-row sample of the classic Iris dataset used to exercise the ingest flow.

## Files
- `data.csv` — tabular data with five columns (sepal_length, sepal_width, petal_length, petal_width, species).
- `schema.json` — JSON Schema describing the CSV fields.
- `metadata.json` — descriptive metadata for the record.
- `checksums.txt` — SHA-256 checksums for integrity verification (generated by `make ingest` prerequisites).

## Notes
- Original dataset: Fisher's Iris (UCI ML Repository); this sample keeps representative rows from all three species.
